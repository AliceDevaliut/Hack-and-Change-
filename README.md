# Описание проекта:
Данный проект представляет собой прототип цифровой образовательной платформы ПСБ, который реализует полный путь преподавателя. Мы разработали две страницы - страницу с аутентификацией пользователя и страницу преподавателя со всеми данными по курсу, учебным материалам, ученикам, их количеству, средней успеваемости. На страницe с курсами есть разделение на 2 блока: "Задания на проверку" и "Мои курсы" У преподавателя есть возможность добавить задание. Он может: добавить название задания, описание к нему, срок сдачи, максимальный балл и прикрепить файл. Также у преподавателя существует возможность добавить курс: установить название, добавить описание, выбрать категорию и добавить изображение, которое будет иллюстрировать курс.
